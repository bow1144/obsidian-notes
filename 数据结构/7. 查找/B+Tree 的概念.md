**B+Tree 和 BTree 的主要差别**：
- B+Tree 的 n 个关键字只对应 n 棵子树
- B+Tree 叶结点**包含全部关键字**，非叶结点的关键字也包含在叶节点
- B+Tree 的非叶结点仅起索引作用，指向子节点的最大关键字
- 一个指针指向最小的叶子结点，所有关键字构成一个线性链表
- 结点个数范围：
	- 非根内部结点：$\lceil m/2 \rceil \leq n \leq m$
	- 根节点：$2 \leq n \leq m$
